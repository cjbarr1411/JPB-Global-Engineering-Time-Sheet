# BizTimeSheet - Weekly Employee Timesheet Tracker

A clean, browser-based time tracking application designed for small businesses. Employees can log daily hours (Sunday to Saturday), view their weekly total instantly, and submit the completed timesheet directly to a manager's email address.

## ✨ Features

- **7-Day Week View** – Week runs from **Sunday to Saturday** (business standard)
- **Instant Total Calculation** – Automatically sums all hours as you type
- **Employee & Recipient Fields** – Enter employee name and manager email
- **Flexible Week Selection** – Pick any start date (Sunday); app auto-corrects invalid dates
- **One-Click Email Submission** – Opens default email client with pre-filled:
  - Employee name
  - Daily breakdown with dates
  - Total weekly hours
- **Responsive Design** – Works on desktop, tablet, and mobile devices
- **Zero Backend Required** – Fully client-side; no database or server needed
